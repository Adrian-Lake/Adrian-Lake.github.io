Geering Up Resource Inventory

Geering Up Resource Inventory (GURI), 2016

Geering Up, a popular STEM summer camp on UBC is required to manage many materials among its staff, GURI is a web based appliation for managing the distrobution and whereabouts of the materials. Written with Node.js it uses MongoDB and Pug templating. Materials are checked out from the store room useing QRCode labels scanned fro the instructors phones, the materials can then be monitored by admin from a interface hosted on Heroku.

Javascript Tutorial

Cloud Break Javascript Tutorial, 2015

In order to provide a quick primer in Javascript for the CodeMakers camp of GeeringUp, a quick turorial was made given a 3 day allotment for curricul planning. The tutorial had to provide a convienet way of executing code and preventing crashes where were expected when loop useage was taught. As such the Javascript tutorial implements web workers to manage execution, in addition templating is used for individual components and checks the evaluation of the code to see if the student has correctly followd instruction.

QtOSA

QtOSA2016

In order to use an outdated, but functionally useful optical spectrum analyser (OSA) housed in the Dr.Madisons QDG lab, a Qt interface was made for OSA connected to a laptop throuhg GPIB. The interface replaces most of the day to day functionality of the physical OSA interface. The program was based useing multi-threading to eliminated crahes and unpredictable behaviours.
